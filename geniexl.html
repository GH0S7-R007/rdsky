How to install GenieXL

Step 1   Connet your field USB in CPU/ Cloud Link
Step 2   Find the system bit and start the installation based on the system bit 32/64

1-install redmon19 as admin
2-install cutewriter as admin
3-install gs922.exe as admin
4-install gxprint1.0 as admin
5-install nxlog-ce-2 as admin




Step 3   Open device and printers


1-Open CutePDF printer properties


2-Change PORT from CPW to RPT1 redirected port.





3-Check the paper size on the physical printer and set the same for CutePDF,(in General and Advance)


4-Then set Cutepdf as default printer



Step 4 open 0.7.1 folder and Run utils (Registerdevice) as admin based on the bit version 32/64


Step 5 Type the pos id e0:5a:1f:8c:0d:a8 in Vb to POS ID button under jarvis Store mgmt. Then the jarvis will assign one device id to you.
After Now run the same file as admin and get confirmation from pc. you can see the confirmation like this picture.


Step 6 install RS Genie XL Aplication.

7-Check the device is online. ( use ping command for that )

Step 8 install UshopgeniexL 3.5.1

1- Select Genie X or XL


2- Select UI or Less

3- Confirm using Hold print


Step 9   Activate test mode

You can able to see 4 options


1-Multiprint-genei name as gxl:// printer name as per physical printer name
2-Printer (swap printer if required)
3-Windows title name------need to find with scan detection tool(scanner1.exe)
4-Anydesk path---set detais.
Deactivate Testmode




Now scan and print then check health...if you need help raise a ticket and post in the group.
