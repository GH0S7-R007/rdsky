Tutorial of Zoho

Learn Zoho

Follow the steps for create a store lead

1. Open Zoho .

2. Open settings (Marked)

3. Select The Store Leads or Partner Stores. .

4. The Store Leads are visible here. Use search button for find the store name from the list. We are selecting the "AP Confectionery store" from the list.

5. Select the store name from the list

6. Check-in at the store ( Mark 1 )

7. Go to details to add all details about the store ( Mark 2)

8. Select the pencil mark for edit.

9. Fill all details and Click on the Tick mark to save.

Owner's first name and last name
Mobile number
Email id
Store status
Address
Pincode
city
Locality
No of billing counters
System OS
Name of the billing software
Printer detils
Barcode scanner name
Store GST number

10. Convert the store to partner store, If owner agreed for instant installation.

11. Open Partner store list to see the store details, Because its converted from the store leads to Partner store

12. Update the Internet status of the store

13. Make sure to create note , what you have done in the store.Touch the + mark near attachment for add note.

14. Touch the tick mart after typing the comment.

15. Your comment has been visible to all now.
