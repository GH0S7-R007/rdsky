How to install GenieX

Step 1   Connect the Genie device on the backside of the CPU
Step 2   Find the bit version(32/64) from the system properties.
Step 3   Connect your field pendrive in CPU and copy the files to desktop based on bit version(32/64)
Step 4   Start the installation

1-open device manager and update CDC serial from file














2-install redmon19 as admin
3-install cutewriter as admin
4-install gs923.exe as admin
5-install gs922.exe as admin
6-install gxprint1.0 as admin
7-install nxlog

Part 2:-View Video

Step 5   Open device and printers for CutePDF configuration


1-Open CutePDF printer properties.

2-Change PORT from CPW3 to RPT1 redirected port.






3-Check the paper size on the physical printer and set the same for CutePDF,(in General and Advance)






3-set Cutepdf as default printer


Step 6 install ushopgeniexL 3.5.1 and activate test mode

You can able to see 4 options


1-Multiprint-genie name as generic text only and printer name as per physical printer name
2-Printer (swap printer if required)
3-Widget-----windows title name------Need to find with the scan detection tool(scanner1.exe)
4-Anydesk path---set detais.

Now deactivate Testmode

Step 7 Map the device to store

Now scan and print, then check health...if you need any help, raise a ticket and post it in the group.
