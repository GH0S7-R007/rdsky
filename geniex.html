How to install GenieX

Step 1   Connect the genie device in back side of the CPU
Step 2   Connect your field USB in CPU
Step 3   Start installation based on system bit 32/64 version

1-open device manager and update CDC serial from file














2-install redmon19 as admin
3-install cutewriter as admin
4-install gs923.exe as admin
5-install gs922.exe as admin
6-install gxprint1.0 as admin

Step 4   Open device and printers


1-Cutepdf printer properties-- select PORT as RPT1 redirected port.






2-Set the paper size as 80 width and 297 height in mm






3-set Cutepdf as default printer


Step 5 install nxlog and ushopgeniexL 3.4.6 . Then activate test mode

You can able to see 4 options


1-Multiprint-genie name as generic text only and printer name as per physical printer name
2-Printer (swap printer if required)
3-Widget-----windows title name------need to find with scan detection tool
4-Anydesk path---set detais.

Now deactivate Testmode



Now scan and print, then check health...if you need any help, raise a ticket and post in the group.
