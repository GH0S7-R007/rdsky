How to do Digital Dashboard Sign-in

Open the browser and type "store.rdskytech.com". Then the page will be open like the below image. Login here through the google account in which mentioned in Zoho.








After login with the google account, the same page visible like the below image. Request OTP to the registered mobile number and start sign in.









Now open the Gmail id and open the message from redsky. then click on the sign in button. Then you can see the page like the image below.









Again open Gmail for the access code. Copy the code and paste it to the above sign-in page. The access code Gmail is visible like the image below.









Put a tick on the box for Agree & Continue. Refer the below picture for clarification.









Scroll down and enter the store owner's name in top and bottom places.Then click Finish. Refer the below image.









DDA completed from the store side. Do the same for your email too. Then DDA process completed.
